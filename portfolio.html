EpicGames

Notre site internet, offre une plateforme interactive pour des jeux en solo ou en équipe en ligne. Les utilisateurs peuvent créer un compte entièrement personnalisé, incluant un avatar de jeu unique. La plateforme propose une vaste sélection de jeux, tels que le poker, des quizz, des courses de voitures, Puissance 4, des jeux de dessin, et bien plus encore. En vous connectant, vous pouvez défier vos amis ou rejoindre des parties avec d'autres joueurs du monde entier, tout en profitant d'une expérience ludique et immersive. Ce site vise à réunir les passionnés de jeux autour d'une interface conviviale et riche en fonctionnalités. Communication : Rest API & SocketIO Pour le backend : Express Pour le frontend : TypeScript/ReactJS Pour la base de données : MariaDB

DashBoard

Notre projet de site internet offre une personnalisation complète de l'écran d'accueil avec une variété de widgets interactifs. Les utilisateurs peuvent configurer des widgets pour suivre la qualité de l’air en entrant le nom de leur ville, obtenir les prévisions météorologiques locales, et afficher des playlists ou des artistes Spotify après s'être connectés à leur compte. De plus, le site permet d'utiliser Google Traduction en entrant le texte souhaité, et de suivre les prix des monnaies virtuelles en entrant le nom de la crypto-monnaie d'intérêt. Cette plateforme vise à offrir une expérience utilisateur personnalisée et intuitive, répondant aux besoins et centres d'intérêt de chaque utilisateur. Les APi utilisée : Météo France, Spotify, Google traduction, Binance Pour le backend : Spring boot Pour le frontend : React Pour la base de données : MariaDB

Wask

Notre application innovante, WASK (Who Ask?), révolutionne la manière dont vous interagissez avec vos amis, à l'image de BeReal. Chaque jour, vous recevez une notification aléatoire vous invitant à vous connecter et choisir votre mode de jeu : répondre à une question, prendre une photo, ou dire une vérité. Une fois votre choix effectué et votre réponse soumise, vous pourrez voir les réponses de vos amis et tenter de deviner qui a répondu parmi votre liste d'amis. WASK transforme les interactions quotidiennes en moments de surprise et de découverte, renforçant les liens avec vos proches de manière ludique et spontanée. Pour le backend : Spring boot Pour le frontend : React Pour la base de données : Mysql

ChatTech

Notre projet est une plateforme de chat moderne permettant aux utilisateurs de créer un compte, de gérer leur profil, et de participer à des discussions individuelles ou en groupe. Ils peuvent former des groupes privés ou publics et partager des images et des GIFs. L'interface est intuitive et fonctionne sur tous les appareils, assurant une expérience utilisateur fluide et agréable. Pour le backend : Node JS Pour le frontend : React Pour la base de données : MongoDB
